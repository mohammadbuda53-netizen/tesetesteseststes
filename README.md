# Handwerker Management App

Eine einfache, lokale Web-App zur Verwaltung von Handwerkern inklusive Status, Priorität, Notizen und Filterung.

## Funktionen

- Handwerker anlegen (Name, Gewerk, Telefon, Status, Priorität, Notiz)
- Bestehende Einträge löschen
- Status und Priorität direkt in der Liste ändern
- Filter nach Gewerk und Status
- Automatische Speicherung im Browser via `localStorage`

## Start

Die App benötigt keinen Build-Prozess.

1. `index.html` im Browser öffnen
2. Einträge erfassen und verwalten

## Dateien

- `index.html` – Struktur und UI
- `styles.css` – Layout und Design
- `app.js` – Logik, Rendering und Persistenz
